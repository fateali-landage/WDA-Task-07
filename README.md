# React Task Manager Application

## Description
A task manager application built using React that allows users to add, edit,
delete, and filter tasks. Tasks are persisted using Local Storage.

## Features
- Task CRUD operations
- Filtering (All, Completed, Pending)
- Local Storage persistence
- Reusable React components

## Technologies
- React
- JavaScript
- CSS
- Local Storage

## Deployment
Deployed using Netlify / Vercel.
